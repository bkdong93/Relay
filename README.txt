Relay 프로젝트를 봐주셔서 감사합니다.

먼저, db 폴더에 있는 ab 폴더를 D드라이브에 옮기신 후
Relay 테이블생성(최종).txt를 Oracle에서 전체 붙여넣기를 하시면 기본데이터 세팅이 완료됩니다.

이후 Relay.zip의 압축을 풀어서 이클립스를 통해 import하시면 실행이 가능합니다.
main page는 WebContent 폴더 밑에 index.jsp 를 실행하시면 됩니다.

혹은 Relay.pdf의 40번째 슬라이드에 있는 http://relaykd.ddns.net:8080/Relay 에 접속하시면
사이트 확인이 가능합니다. ( 조원 백경동이 집에서 서버를 켜두었을 경우에만 해당 )
